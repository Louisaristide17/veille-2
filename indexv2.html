🌐 Technologies Web et Langages de Programmation

📌 Technologie Web

Une technologie web désigne l’ensemble des outils, méthodes, protocoles et logiciels utilisés pour concevoir, développer et faire fonctionner des sites et applications accessibles via Internet ou un réseau local. Elle comprend le HTML, le CSS, le JavaScript, les serveurs web, les bases de données et les frameworks.

💻 Langage de Programmation Web

Un langage de programmation web est un langage informatique utilisé pour créer des sites et applications web. Il peut fonctionner côté client (comme JavaScript) ou côté serveur (comme PHP, Python, Node.js). Il permet de rendre les pages dynamiques et interactives.

📱 Langage de Programmation Mobile

Un langage de programmation mobile est utilisé pour développer des applications destinées aux smartphones et tablettes. Par exemple : Java et Kotlin pour Android, Swift pour iOS, ou Flutter pour les applications multiplateformes.
